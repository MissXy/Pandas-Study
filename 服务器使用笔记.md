## 服务器使用命令

#### 文件夹操作

```shell
# 列出当前文件夹
ls
# 新建文件夹 test
mkdir test
# 删除文件夹 test
rmdir test
```

#### 查看配置信息

```shell
# 显示CPU信息
cat /proc/cpuinfo
# 查看CPU使用情况
top
# 查看GPU使用情况
nvidia-smi
# 实时查看，1秒刷新1次
watch -n 1 nvidia-smi

# 单独查看内存使用情况
free -m
```

#### 本地文件上传至远程服务器`MobaXterm`

1. 使用`MobaXterm`软件，上传或者传输，只需要**手动拖拽至左边工作区**即可。

2. 运行服务器端 `*.py` 文件

```shell
# 文件名为:test.py, 终端输入
python3 ./test.py
```

3. 可以通过左边工作区打开多个终端
